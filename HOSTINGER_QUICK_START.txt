========================================
TROUBLESHOOTING
========================================

If deployment fails:
1. Check DATABASE_URL is exactly right (username, password, database name)
2. Make sure SESSION_SECRET is set
3. Verify database exists in Hostinger (u210215546_estimax)
4. Check Hostinger logs for specific error messages

Common mistakes:
❌ Wrong password in DATABASE_URL
❌ Missing @ or / in DATABASE_URL
❌ Using wrong database name
❌ Forgetting to set SESSION_SECRET
